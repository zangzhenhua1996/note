    Urllib库¶

    内置的HTTP请求库¶

    • 请求模块： urllib.request
    • 异常处理模块: urllib.error
    • url解析模块:urllib.parse
    • robots.txt解析模块:urllib.robotparser
    • 代理: ProxyHandler (翻墙)
    • Cookie: http.cookiejar
    • 网址拼接: urljoin
    • 将字典转换成请求参数: urlencode
